## 每日一道编程题（350）：中南大学上机题（二）
在太平洋的一个小岛上，岛民想要建立一个环岛的堤坝，我们可以将小岛简化为一个二维平面，你需要使用K条边（这些边要么是水平或者垂直长度为1的边，要么是倾斜45度的长度为√2的边）围城一个多边形，多边形的顶点必须位于整点，然后要让围成的多边形面积最大，你需要求出最大面积是多少。

**输入格式：**
输入包含多个测试实例，每组实例给出一个数K（3≤k≤2,000,000,000）

**输出格式：**
每一组对应一个要求的答案（保留一位小数）

**输入样例：**
``` c
3
4
5
6
```
**输出样例：**
``` c
0.5
2.0
2.5
4.0
```
